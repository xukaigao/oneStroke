# ✏️ 一笔画 One-Stroke

笔不离纸，把图形的**每条线都走一遍**，而且**每条只走一次**。纯 HTML + CSS + JavaScript，无任何依赖，适合 6~12 岁孩子。

在线试玩：<https://xukaigao.github.io/oneStroke/>

## 怎么玩

1. 屏幕上是由**圆点（节点）+ 灰色线（边）**组成的图形。
2. 先点一个圆点开始，然后**按住拖到相邻的圆点**，沿着已有的线走；走过的线会变成彩色。
3. 不能重复走同一条线；把**所有线都走完**就一笔画成 ✨。
4. 走不通了就「撤销」回退几步，或「重来」从头开始。「提示」会帮你高亮起点或下一步。
5. 通关后会弹出鼓励语，点「下一图」继续挑战。

## 一笔画的小数学

把图形看成「点 + 线」的网络，看每个点连了几条线（度数）：

| 奇数度的点 | 能否一笔画成 | 起笔 |
| --- | --- | --- |
| 0 个（全偶数） | 能，且能回到起点 | 任意点 |
| 恰好 2 个 | 能 | 必须从其中一个奇数点起，另一个收笔 |
| 多于 2 个 | 不能 | — |

> 比如「田字」有 4 个奇数点，所以**画不成**；「日字」「信封」「五角星」都能一笔画成。

## 关卡来源

- **24 个固定图形**（全部手工定义、已确保可一笔画成），用顶部**下拉框**选图，也可用「上一图 / 下一图」翻页。
- 包含：三角形、蝴蝶结、五角星 ★、方块斜线、日字、风筝、信封 ✉️、斜角双格、四个圆、五边套菱、折角的纸、三角配圆、圆心双三角、三角套 X、8 字、太极、花朵、六芒星、风车、钥匙、字母 B、圆里折线、圆内三角、圆里弦弧。
- 直线与圆弧混合，同一对端点之间可以有多条边（如一个圆的两段弧）。

## 实现说明

- 坐标基于格点（lattice），节点与边都用字符串 key 表示，便于判重。
- **欧拉算法（Hierholzer）** 用于：校验某图是否可一笔画成、计算合法起点、以及「提示」时从当前笔尖位置求出可行的下一步。
- 渲染用 SVG，已画的边按顺序生成彩虹色轨迹；指针事件统一处理触摸与鼠标，移动端可直接手指拖。

## 本地运行

直接用浏览器打开 `index.html` 即可（或起一个静态服务器）：

```bash
python -m http.server 8000
# 然后访问 http://localhost:8000
```
